## Key Insights

- The business maintained around 40% profit margin across the years.
- Returning customers contribute significantly to total revenue.
- Some months show higher demand for lower-priced products, affecting profit trends.
- Payment preferences change over time.
- Delivery and return analysis highlights operational performance areas.
