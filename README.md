# Amazon Sales Dashboard (Power BI)

This project presents a 3-year analysis (2023–2025) of Amazon sales data using Power BI.

The objective of this dashboard is to transform raw sales data into meaningful business insights by analyzing sales performance, product trends, customer behavior, profit margins, payment patterns, and delivery performance.

---

This dashboard provides a complete business view including:

- Overall Sales and Profit Performance
- Product-Level Analysis
- Customer Behavior Insights
- Yearly and Monthly Sales Trends
- Payment Mode Analysis
- Delivery and Return Analysis

It helps understand how the business is performing and where improvements can be made.

---

## Key Insights

- The business maintained around 40% profit margin across the years.
- Returning customers contribute significantly to total revenue.
- Some months show higher demand for lower-priced products, affecting profit trends.
- Payment preferences change over time.
- Delivery and return analysis highlights operational performance areas.
